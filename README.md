# IoT-Based Vehicle Accident Detection & Emergency Alert System

## Overview

A Final Year B.Tech Project developed using ESP32, GPS, MPU9250, LoRa, Telegram Bot and SMS APIs for accident detection and emergency alerting.

## Features

* Accident Detection
* GPS Tracking
* Telegram Alert
* SMS Alert
* LoRa Communication
* False Alert Prevention
* Real-Time Monitoring

## Hardware Used

* ESP32
* MPU9250 / MPU6500
* NEO-6M GPS
* LoRa SX1278
* OLED Display
* Buzzer

## Technologies

* Embedded C
* Arduino IDE
* IoT
* GPS
* LoRa
* Telegram Bot API
* Fast2SMS API
